# Refonte visuelle HealSpace

## Changements principaux

- Suppression du logo « H » dans la barre supérieure et sur la page de connexion.
- Nouvelle palette désaturée bleu-gris, sauge et ardoise.
- Suppression des grands dégradés bleu électrique / violet.
- Cartes, boutons, champs et menus plus sobres.
- Ombres plus légères et bordures plus naturelles.
- Navigation, fil d’actualité, profil, messagerie et modales harmonisés.
- Responsive mobile, tablette et ordinateur conservé.
- Mode sombre adapté à la nouvelle identité.
- Respect de `prefers-reduced-motion` pour l’accessibilité.

## Lancement

```bash
npm install
npm run dev
```

## Vérification production

```bash
npm run build
```
